# 00 总览 —— 这套文档讲什么

本目录归档了一段"在 WSL 里从零开始把 MNN 端侧大模型跑到安卓手机上"的实操记录。

## 一句话总结整体流程

```
WSL 装 JDK + cmdline-tools + NDK + CMake
        ↓
编 MNN Android 核心库（生成 libMNN.so 等）
        ↓
编 MnnLlmChat App（生成 APK）
        ↓
通过 adb（推荐 WiFi 无线调试）装到手机
```
