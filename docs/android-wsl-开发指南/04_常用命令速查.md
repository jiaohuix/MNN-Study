# 04 常用命令速查

> 把日常开发会用到的命令收在一起，方便随时翻。

## adb 设备管理

```bash
adb devices                                   # 列出已连接设备
adb kill-server && adb start-server           # adb 卡住时重启
adb pair  <ip>:<pair_port>                    # 无线调试配对（首次）
adb connect <ip>:<port>                       # 无线调试连接
adb disconnect                                # 断开所有无线设备
adb -s <serial> <command>                     # 多设备时指定目标
```

## APK 安装与卸载

```bash
adb install app-debug.apk                     # 安装
adb install -r app-debug.apk                  # 覆盖安装（保留数据）
adb install -t app-debug.apk                  # 允许 test-only APK
adb uninstall com.alibaba.mnnllm.android      # 卸载（按包名）
adb shell pm list packages | grep mnn         # 查看 mnn 相关已安装包
```

## 启动 / 停止 App

```bash
# 启动 demo 的某个 Activity
adb shell am start -n com.taobao.android.mnndemo/.VideoActivity

# 停止 App
adb shell am force-stop com.alibaba.mnnllm.android
```

## 日志（logcat）

```bash
adb logcat                                    # 实时全量日志
adb logcat | grep -i mnn                      # 只看 mnn 相关
adb logcat -c                                 # 清空日志缓冲
adb logcat -s MNNJNI:V Chat:V                 # 按 tag 过滤
adb logcat *:E                                # 只看 Error 级别
```

## 文件传输

```bash
adb push localfile /sdcard/Download/          # 电脑 → 手机
adb pull /sdcard/Download/foo.bin ./          # 手机 → 电脑
adb shell ls /sdcard/                         # 查看手机文件
```

## Gradle 构建

```bash
./gradlew tasks                               # 列出所有任务
./gradlew assembleDebug                       # 编 debug APK
./gradlew assembleRelease                     # 编 release APK
./gradlew installDebug                        # 编 + 装
./gradlew clean                               # 清理产物
./gradlew --stop                              # 停掉 gradle daemon（卡时用）
./gradlew assembleDebug --info                # 详细日志（debug 编译失败时用）
./gradlew assembleDebug --stacktrace          # 出错时打印 Java 堆栈
```

## sdkmanager

```bash
sdkmanager --list                             # 列出可装/已装组件
sdkmanager --list_installed                   # 只看已装
sdkmanager --update                           # 更新已装组件
sdkmanager --install "ndk;26.1.10909125"      # 装某个组件
sdkmanager --uninstall "ndk;<version>"        # 卸载
```

## CMake / NDK 直编（不走 gradle）

```bash
# project/android/build_64.sh 内部其实就是这套调用
cmake .. \
  -DCMAKE_TOOLCHAIN_FILE=$ANDROID_HOME/ndk/<ver>/build/cmake/android.toolchain.cmake \
  -DANDROID_ABI=arm64-v8a \
  -DANDROID_PLATFORM=android-21 \
  -DANDROID_STL=c++_shared
make -j$(nproc)
```

## 看 App 在手机上的私有目录

```bash
# 需要 root 或 debug 包才能进
adb shell run-as com.alibaba.mnnllm.android ls files/
adb shell run-as com.alibaba.mnnllm.android cat files/log.txt
```

## 抓性能 / 内存

```bash
adb shell top -m 10                           # 实时进程 CPU
adb shell dumpsys meminfo com.alibaba.mnnllm.android   # 内存详情
adb shell dumpsys gfxinfo com.alibaba.mnnllm.android   # 渲染性能
```

## 一键脚本：编 + 装 + 看日志

放到 `~/bin/mnn-run` 之类的位置：

```bash
#!/usr/bin/env bash
set -e
cd ~/Projects/nlp/MNN/apps/Android/MnnLlmChat
./installDebug.sh
adb logcat -c
adb shell am start -n com.alibaba.mnnllm.android/.MainActivity
adb logcat | grep -i -E "mnn|chat"
```
